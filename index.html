Why Zoi? GitHub

Ultra-Simple Zig Server

Zoi delivers content over http 1.1 without relying on any external dependencies other than the standard library. All you need is the Zoi source code, Zig, and the Zig standard library. Zoi is released under the MIT license.To deliver static content just put some html/css/js files in the directory where you run Zoi and it will automatically serve any files it finds. Dynamic content is also possible with a little bit of coding. Zoi expects an index.html file and a 404.html file to be present at minimum when using static content delivery.
